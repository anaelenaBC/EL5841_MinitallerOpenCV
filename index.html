Bibliotecas embebidas para visión por computador

OpenCV y sus aplicaciones en inteligencia artificial

Conocimiento previo

Una librería en software es un conjunto de códigos, funciones, clases o módulos predefinidos que los programadores pueden usar para realizar tareas comunes sin tener que escribir todo desde cero. Es, básicamente, un paquete de herramientas listas para usar que acelera el desarrollo y mejora la eficiencia del código.

Una librería embebida es una librería que viene incorporada dentro de un software o sistema, de manera que no necesitas instalarla por separado. Se dice que está "embebida" porque forma parte del paquete principal o del entorno de ejecución, garantizando compatibilidad y facilidad de distribución.

La visión por computadora es un área de la inteligencia artificial que permite a las computadoras "ver" y entender imágenes o videos de manera similar a como lo hace un ser humano. Su objetivo es que una máquina pueda interpretar información visual y tomar decisiones o generar acciones basadas en ella.

OpenCV (Open Source Computer Vision Library) es una librería de software de código abierto diseñada para visión por computadora y procesamiento de imágenes. Es muy popular porque permite que los desarrolladores construyan aplicaciones que puedan "ver" y entender imágenes y videos de manera eficiente y robusta.

Teoría

¿Qué es la visión por computador y OpenCV?

Visión por computador es una rama de la inteligencia artificial que permite a las máquinas "entender e interpretar imágenes y vídeos", extrayendo información de su contenido. Esto abarca tareas como reconocer objetos, detectar rostros, analizar movimientos o incluso conducir vehículos de forma autónoma. Un ejemplo común es la detección de objetos en imágenes: el sistema identifica y delimita elementos como personas, animales o artículos cotidianos en una escena. Gracias a técnicas de este tipo, un computador puede "ver" una imagen o video y tomar decisiones basadas en lo que observa, imitando en cierta medida la visión humana.


OpenCV (Open Source Computer Vision Library) es una biblioteca open source diseñada justamente para facilitar el desarrollo de aplicaciones de visión por computador. Contiene cientos de algoritmos de procesamiento de imágenes y video listos para usar, lo que la ha convertido en una herramienta líder y estándar en el campo. De hecho, OpenCV incluye más de 2500 algoritmos optimizados los cuales permiten resolver problemas variados de visión artificial: detección de objetos, clasificación de imágenes, segmentación de regiones, reconocimiento facial, seguimiento de movimientos, reconstrucción 3D.


OpenCV se caracteriza por su amplio ecosistema y portabilidad. Es multiplataforma (funciona en sistemas operativos de escritorio como Windows, Linux, macOS, y en móviles Android o iOS) y multilenguaje (ofrece interfaces en C++, Python, Java, Matlab, etc.), lo cual la hace muy accesible tanto en proyectos industriales como académicos. Además, está escrita principalmente en C/C++ altamente optimizado, aprovechando instrucciones SIMD (como SSE/NEON) y pudiendo utilizar aceleración por hardware (GPU) mediante tecnologías como CUDA y OpenCL. Gracias a esta eficiencia, OpenCV permite procesamiento en tiempo real, cualidad fundamental en aplicaciones embebidas que deben analizar video en vivo (por ejemplo, visión en robots, drones o sistemas de asistencia vehicular). En resumen, OpenCV proporciona un "toolkit" esencial para incorporar visión por computador en proyectos embebidos de ingeniería electrónica, al abstraer la complejidad de los algoritmos de visión en funciones listas para usarse.

Estructura modular de OpenCV

OpenCV está organizada en módulos, lo que permite incluir solo lo necesario según la aplicación y mantener la biblioteca eficiente y flexible. Entre los principales se destacan:

  • Core (core): Núcleo básico. Define estructuras de datos (como cv::Mat) y operaciones matemáticas esenciales.
  • Procesamiento de Imágenes (imgproc): Funciones clásicas como filtrado, cambios de color, transformaciones geométricas y detección de bordes.
  • Códecs de Imágenes (imgcodecs): Lectura y escritura de formatos comunes (PNG, JPEG, BMP).
  • Entradas/Salidas de Video (videoio): Captura desde cámaras o archivos y grabación de videos.
  • Interfaz Gráfica (highgui): Herramientas simples para mostrar imágenes, crear ventanas y capturar eventos de teclado.
  • Análisis de Video (video): Métodos para detectar movimiento, sustracción de fondo y seguimiento de objetos.
  • Calibración y 3D (calib3d): Calibración de cámaras, estimación de pose y reconstrucción tridimensional.
  • Características 2D (features2d): Detección y emparejamiento de puntos de interés (SIFT, ORB, Harris, etc.).
  • Detección de Objetos (objdetect): Algoritmos como Viola-Jones (caras, ojos, peatones).
  • Redes Neuronales (dnn): Carga e inferencia de modelos entrenados en TensorFlow, PyTorch, ONNX, etc.
  • Aprendizaje Automático (ml): Algoritmos clásicos como SVM, árboles de decisión, clustering, K-means.
  • Fotografía Computacional (photo): Técnicas de mejora y restauración de imágenes (denoising, HDR, inpainting).
  • Composición de Imágenes (stitching): Creación automática de panoramas a partir de fotos superpuestas.
  • Módulos auxiliares: FLANN, pruebas unitarias y bindings para distintos lenguajes.

Demostración práctica

1_reconocimientoColores.py

Este programa utiliza OpenCV para detectar y reconocer colores específicos en tiempo real a través de la cámara web. Implementa algoritmos de segmentación de color en el espacio HSV para identificar objetos de colores predefinidos y dibujar contornos alrededor de ellos.

2_direccionMirada.py

Este programa utiliza OpenCV para detectar en tiempo real si el usuario está mirando directamente a la cámara web. Emplea algoritmos de detección facial y de ojos para analizar la posición de los ojos dentro del rostro y determinar la dirección de la mirada. Si ambos ojos son detectados y están alineados hacia la cámara, el sistema muestra un mensaje indicando que el usuario está prestando atención. Esta funcionalidad es especialmente útil en aplicaciones de seguridad o de control de acceso.

3_filtroLentes.py

Este programa implementa un filtro de realidad aumentada que superpone lentes virtuales sobre el rostro del usuario en tiempo real utilizando la cámara web. El proceso comienza detectando la cara y los ojos mediante los clasificadores Haar Cascade de OpenCV. Una vez localizados los ojos, el sistema calcula la posición y el tamaño adecuados para colocar la imagen de los lentes virtuales. Esta técnica es ampliamente utilizada en aplicaciones de entretenimiento, redes sociales y demostraciones de realidad aumentada.

4_separarCaras.py

Este programa utiliza OpenCV para analizar imágenes y determinar si contienen rostros humanos. Emplea algoritmos de detección facial que examinan cada imagen de una carpeta de entrada, identificando la presencia o ausencia de caras. Si se detecta al menos un rostro en la imagen, el archivo se mueve automáticamente a una carpeta designada para imágenes con caras; en caso contrario, se traslada a una carpeta separada para imágenes sin caras. Este proceso es útil para organizar grandes volúmenes de fotografías.

Tutorial de instalación

Instalación de Python

1
Visitar https://www.python.org/downloads/
2
Descargar la última versión (recomendada)
3
Ejecutar el instalador y seguir los pasos del asistente
4
Verificar que Python está instalado correctamente
5
Abrir una terminal y ejecutar el siguiente comando:
python --version
6
Si aparece la versión de Python, la instalación fue exitosa
7
Si no aparece la versión, es necesario configurar las variables de entorno
8
Abrir el Panel de Control de Windows y buscar "Variables de entorno"
9
En las variables de entorno, buscar la variable "Path" y agregar las rutas de Python
10
Agregar las siguientes rutas:
C:\Users\Usuario\AppData\Local\Programs\Python\Python313\ C:\Users\Usuario\AppData\Local\Programs\Python\Python313\Scripts\
Nota
Reemplazar "Usuario" con su nombre de usuario real en las rutas de Python. Después de modificar las variables de entorno, es necesario cerrar y abrir la terminal para que los cambios surtan efecto.

Instalación de OpenCV

1
Abrir una terminal o línea de comandos
2
Ejecutar el siguiente comando para instalar OpenCV:
pip install opencv-python
3
Verificar que OpenCV se instaló correctamente:
python -c "import cv2; print(cv2.__version__)"
4
Si aparece la versión de OpenCV, la instalación fue exitosa
5
Si no aparece la versión, configurar las variables de entorno con la ruta de OpenCV
6
Agregar la siguiente ruta a las variables de entorno:
C:\Users\Usuario\AppData\Local\Programs\Python\Python313\Lib\site-packages\opencv_python_headless-4.11.0.86\opencv_python_headless.cp313-win_amd64.pyd
